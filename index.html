Functions

new (options)
Constructor function of UberScore module

Parameters:

  • options

    a table with options for the score counter curve

    • x: (number) the x position of the score counter (default 0)
    • y: (number) the y position of the score counter (default 0)
    • score: (number) the initial score value (default 0)
    • maxDigits: (number) how many digits the score counter does have (default 6)
    • leadingZeros: (number) how many leading zeros the score counter does have (default 6)
    • fontName: (string) the name of the bitmap font
    • fontSize: (number) the size of the bitmap font (default 64)
    • tracking: (number) the distance between single digits (default 32)
    • animation: (number) the animation type (0=off,1 or 2) (default 0)
    • animationSpeed: (number) the transition speed of the animation in millisecs (default 250)
    • transitionIn: (transition) custom easing transitions when a new digit comes in (default easing.linear)
    • transitionOut: (transition) custom easing transitions when a new digit goes out (default easing.linear)
    • updateFrequency: (number) the frequency of the score counter updates (default 100)
    • instantChange: (bool) if the whole score buffer should be instantly added to the counter on next update (default false)

Usage:

      local UberScore = require("devilsquid.uberscore.uberscore")
 	local score = UberScore:new()

  • freq number the frequency of the updates in millisecs
uberScore:CreateAction (options)
Add actions that get fired when a certain values gets added or is reached

Parameters:

  • options

    a table with options for the action

    • value: (number) the value where the action gets triggered
    • trigger: (string) the kind of trigger (default “above”) match = exactly match the value above = exactly match or above the value below = exactly match or below the value (for negative score counter) add = if an exact value gets added addAtLast = if an exact or bigger as value gets added sub = if an exact value gets subtracted subAtLast = if an exact or bigger as value gets subtracted
    • pfx: (string) the name of the particle fx in the pfx folder (default “stars”)
    • duration: (number) the duration of the particle effect (default 1000)
    • iteration: (number) the kind of trigger (default 1)
    • callback: (function) a custum callback function

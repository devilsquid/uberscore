new (options)
Constructor function of UberScore module

Parameters:

  • options

    a table with options for the score counter

    • x: (number) the x position of the score counter (default 0)
    • y: (number) the y position of the score counter (default 0)
    • score: (number) the initial score value (default 0)
    • maxDigits: (number) how many digits the score counter does have (default 6)
    • leadingZeros: (number) how many leading zeros the score counter does have (default 6)
    • fontName: (string) the name of the bitmap font
    • fontSize: (number) the size of the bitmap font (default 64)
    • tracking: (number) the distance between single digits (default 32)
    • animation: (number) the animation type (0=off,1 or 2) (default 0)
    • animationSpeed: (number) the transition speed of the animation in millisecs (default 250)
    • transitionIn: (transition) custom easing transitions when a new digit comes in (default easing.linear)
    • transitionOut: (transition) custom easing transitions when a new digit goes out (default easing.linear)
    • updateFrequency: (number) the global frequency of the score counter updates (default 100)
    • instantChange: (bool) if the whole score buffer should be instantly added to the counter on next update (default false)

Returns:

    returns an UberScore object

Usage:

     local UberScore = require("devilsquid.uberscore.uberscore")
 local score = UberScore:new()
uberScore:Add (value[, step=1[, freq]])
Add a value to the score counter

Parameters:

  • value number the value to add to the score counter
  • step number how much of the value should be added per update (default 1)
  • freq number the frequency of the updates in millisecs (optional)

Returns:

    nothing

Usage:

     -- add 15 pounts to the counter
 score:Add( 15 )

 -- add 15 points in three steps which leads to adding 5 points per update cycle
 -- use this for big numbers that has to be counted fast
 score:Add( 15, 3 )

 -- add 15 points in one step (default) with an update speed of 25 millisecs
 -- if you set up an UberScore counter with a global update frequency of 100 millisecs
 -- you can add these 15 points with a higher speed to the counter
 -- this is intended to give player feedback (fast score counting = the player achieved something special)
 score:Add( 15, 1, 25 )
uberScore:CreateAction (options)
Add actions that get fired when a certain value gets added or is reached

Parameters:

  • options

    a table with options for the action

    • value: (number) the value where the action gets triggered
    • trigger: (string) the kind of trigger (default “above”)
      You can use the following triggers:
      • match = exactly match the value
      • above = exactly match or above the value
      • below = exactly match or below the value (for negative score counter)
      • add = if an exact value gets added
      • addAtLast = if an exact or bigger as value gets added
      • sub = if an exact value gets subtracted
      • subAtLast = if an exact or bigger as value gets subtracted
    • pfx: (string) the name of the particle fx in the pfx folder, you can use “stars” or “hearts” (default “stars”)
    • duration: (number) the duration of the particle effect (default 1000)
    • iteration: (number) the kind of trigger (default 1)
    • callback: (function) a custum callback function

Returns:

    nothing

Usage:

     -- create a built-in stars particle effect when the score reaches 100 points or above
 score:CreateAction({ value=100 })

 -- create a built-in hearts particle effect when the score reaches exactly 500 points or above
 score1:CreateAction({ value=200, trigger="match", duration=500, pfx="hearts"  })

 -- create a custom particle effect when more than 74 points gets added to the counter
 local function EmitParticleFX()
     local pfx = ParticleDesigner:new({ fileName = "my/particle/effect/file.json", duration=500 })
     pfx.x = score.x
     pfx.y = score.y
     local width = score:GetWidth()
     transition.to( pfx, {time=500, x=pfx.x-width})
 end
 score:CreateAction({ value=75, trigger="addAtLast", callback=EmitParticleFX })
